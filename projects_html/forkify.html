Forkify Web App

JavaScript, HTML, and CSS

Course Instructor & Designer ‐ Jonas Schmedtmann

The app follows the Model View Controller pattern. Recipes can either be fetched from the forkify API or created manually. The user can bookmark each recipe and adjust the serving size as needed. Bookmarked recipes then appear in the bookmarks drop-down menu.
